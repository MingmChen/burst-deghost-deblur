## 3、canny参数设置

关于2个阈值参数：

1. 低于阈值1的像素点会被认为不是边缘；

2. 高于阈值2的像素点会被认为是边缘；

3. 在阈值1和阈值2之间的像素点,若与第2步得到的边缘像素点相邻，则被认为是边缘，否则被认为不是边缘。
